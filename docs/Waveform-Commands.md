### WAVEFORM UNINSTALL

Uninstall waveform.

```
C[D]<seq_number>|waveform uninstall <waveform_name>
```

**Parameters:**
- `<waveform_name>` = name of waveform to uninstall

**Example:**
```
C270|waveform uninstall custom_waveform
```
