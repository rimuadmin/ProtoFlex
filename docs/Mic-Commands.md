### MIC ACC

Set ACC (Accessory) input.

```
C[D]<seq_number>|mic acc <0|1>
```

**Parameters:**
- `<0|1>` = disable/enable ACC input

**Example:**
```
C300|mic acc 0
```

**Response:**
```
R300|0||                                         (success - ACC disabled)
```

**Response Codes:**

| Hex Code | Meaning |
|----------|----------|
| 00000000 | OK |
| 5000002C | Incorrect number of parameters |
