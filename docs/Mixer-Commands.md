### FRONT_SPEAKER

Control front speaker settings.

#### MUTE

Mute/unmute front speaker.

```
C[D]<seq_number>|mixer front_speaker mute <0|1>
```

**Parameters:**
- `<0|1>` = unmute/mute

**Example:**
```
C310|mixer front_speaker mute 0
```
