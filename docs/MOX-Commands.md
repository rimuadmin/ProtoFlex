### XMIT

Enable/disable MOX (Manual On/Off).

```
C[D]<seq_number>|xmit <0|1>
```

**Parameters:**
- `<0|1>` = disable/enable MOX

**Example:**
```
C280|xmit 1
```

**Response Codes:**

| Hex Code | Meaning |
|----------|----------|
| 00000000 | OK |
| 5000002C | Incorrect number of parameters |
| 5000003D | Transmit not supported in this radio |
